Open Events

Regionale Termine aus ganz Deutschland
kostenfrei - offen für alle - open source

Kostenlose Nutzung

Das Eintragen und Abrufen von Terminen soll für die Nutzer kostenfrei sein.

Freie Schnittstelle

Jeder kann Termine über unsere offene JSON-API-Schnittstelle abrufen und/oder neue Termine übermitteln.

Open Source Technik

Unsere Platform basiert auf moderner Open-Source-Technik: Docker, Elasticsearch und Node.js.

Hintergrundstory

Es fing an damit, dass es in Münster relativ schwer ist, einen kompletten Überblick über aktuelle Freizeitaktivitäten zu bekommen. Z.B. Stadtfeste, Flohmärkte, Parties, ... Alles ist über zig verschiedene Internetseiten verteilt. Ständig hat man etwas verpasst.

Bei Gesprächen mit verschiedenen Menschen aus anderen Regionen stellte sich heraus, dass dieses Problem überregional auch in anderen Städten besteht.

Lösung: Eine Internetportal, das alle Termine sammelt

Daher entstand bei CodeForMünster die Idee, eine Opensource Termindatenbank zu bauen, in die jeder kostenlos Termine einpflegen und auslesen kann. Über eine einfach zu nutzende offene JSON-API ermöglichen wir die Nutzung für eine große Bandbreite an Zielgruppen, vom Web-Profi, über Designer bis zum Hobby-Programmierer. Jeder kann sich seine eigene Termin-Seite bauen. Sei es ein Stadtportal oder eine Termin-Liste für die Homepage.

Mitmachen / Kontakt

Wir freuen uns über jeden, der uns bei der technischen und inhaltlichen Umsetzung unseres Projekts helfen möchte. Wir nutzen zur Kommunikation einen Slack-Channel. Du musst nicht im Münsterland wohnen, um mitzuhelfen. Wenn Du mitmachen möchtest, sprich uns über unseren Slack Channel von CodeForMünster an: Anmelden zum Slack Channel von CodeForMünster Du kannst natürlich auch einfach unseren SourceCode auf GitHub auschecken, um am Projekt mitzuprogrammieren. Zur GitHub Seite des Open Events Projekts Oder besuche uns auf einem unserer Treffen. Der nächste Termin wird auf der CodeForMünster-Homepage angekündigt: Zur Homepage von CodeForMünster
